## purchase_manger
- NestJS with GraphQL

##
- NestJS, NodeJS, typescript, GraphQL, MYSQL
- TypeORM, JWT, esLint, Prettier, Swagger
- MacOS (M2), VS code, Postman, Github

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
