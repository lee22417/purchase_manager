## purchase_manger

- NestJS with GraphQL

##

- NestJS, NodeJS, typescript, GraphQL, MYSQL
- TypeORM, JWT, esLint, Prettier, Swagger
- MacOS (M2), VS code, Postman, Github

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## GraphQL

```bash
http://localhost:3000/graphql
```

## Public

```bash
http://localhost:3000/resource/db/user_purchase_status_column.txt
```
